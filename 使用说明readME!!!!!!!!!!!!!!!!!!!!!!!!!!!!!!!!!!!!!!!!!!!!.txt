编译说明：
1，先打开FlatFish.sln,编译后，会生成bin 文件夹；
2，把configfile里的configfilenew.rar文件解压缩(解压到当前文件夹)，复制到bin/Debug64/ 或者bin/Debug32/ 目录下 ；
3，若用到opencv，把opencv2410.rar解压到当前文件夹,否则把用到的工程先移除解决档案；
4，重新编译运行；
5，上传代码前，先pull代码，再push；
注意：注意32位和64位的放到不同文件夹；
